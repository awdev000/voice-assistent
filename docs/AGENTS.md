# AGENTS.md

## Назначение
Проект представляет собой модульную голосовую платформу на Raspberry Pi.

## Принципы разработки
- Не ломать существующую архитектуру без явной необходимости.
- Не добавлять тяжёлые зависимости без обоснования.
- Учитывать ограниченные ресурсы Raspberry (CPU, RAM).
- Поддерживать fallback-архитектуру (Cloud → Local).

## Ограничения
- Основной язык — Python.
- Развёртывание — Docker Compose.
- Локальные модели должны работать на CPU.
- Аудио работает через PipeWire.

## Цель развития
Создать расширяемый голосовой хаб для домашней автоматизации.
