# ONBOARDING

## Для разработчиков

1. Установить Docker и Docker Compose.
2. Клонировать репозиторий.
3. Создать .env файл.
4. Запустить docker-compose.

## Переменные окружения

OPENAI_API_KEY=
MODEL=gpt-4o-mini
HF_TOKEN=

## Основные файлы
- main.py — точка входа
- ai.py — логика ИИ
- tts.py — озвучивание
- audio.py — запись
