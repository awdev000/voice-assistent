# CODE STYLE

## Язык
Основной язык: Python 3.11+

## Принципы
- Чистая архитектура
- Минимальная связанность модулей
- Dependency injection где возможно
- Никакой логики в main.py

## Стиль
- PEP8
- Явные импорты
- Минимум глобальных переменных
- Все магические числа выносить в конфиг

## Логирование
- Использовать logging, не print
- Уровни: INFO, WARNING, ERROR

## Запрещено
- Хардкод путей
- Блокирующие вызовы без timeout
- Прямой вызов LLM без router
