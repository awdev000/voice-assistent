# SKILLS CONCEPT

## Что такое Skill

Skill — это независимый модуль, который:

- определяет, может ли он обработать запрос
- выполняет детерминированное действие
- возвращает текст ответа

## Пример

Skill: Таймер
Вход: "поставь таймер на 5 минут"
Выход: "Таймер установлен на 5 минут"

## Принцип работы

Router проверяет:
1. Может ли Skill обработать текст
2. Если да — вызывает Skill
3. Если нет — передаёт в AI Engine
